Sheepshear

SheepShear, an advanced PowerPC emulator designed to run MacOS classic

Download this project as a .zip file Download this project as a tar.gz file

Project Goals

SheepShear is a fork of the excellent SheepShaver emulator to ensure its continued existence and development.

Target platforms

  • Haiku
  • Linux
  • MacOS X
  • Windows

Compiling

Binary packages aren't available at this time given the early state of SheepShear and its rapid pace of development.

Haiku

$ git clone https://github.com/kallisti5/sheepshear.git
$ cd sheepshear/src/Haiku
$ make

Linux

$ git clone https://github.com/kallisti5/sheepshear.git
$ cd sheepshear/src/Unix
$ ./autogen.sh
$ make

MacOS X

Use Xcode, in a rough state currently as OS X now uses clang by default (which doesn't play nicely with the current PowerPC emulation code)
